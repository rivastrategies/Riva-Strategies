1. EXECUTIVE SUMMARY

This project upgrades the existing flare tip on flare stack MS-468P at the INEOS Polypropylene plant (Battleground Manufacturing Complex, La Porte, TX). A temporary flare is installed first to allow the existing flare to be taken offline for tip replacement and associated modifications. The scope spans two phases: Temporary Flare and Permanent Flare Modification. All drawings are Rev. 0, Issued for Construction 03/11/2026.

Primary Work Scope

  • Replace existing flare tip with new MS-2766P unit at top of flare stack MS-468P
  • Install new 2" staged steam line (Lines 50315 / 50240) from grade to new flare tip — 4+ iso sheets
  • Replace Flame Front Generator (FFG) and all connected fuel gas / AI piping (Lines 41001, 41005, 41006, 41007, 41008, 41009)
  • Install temporary 24" flare piping (TEMP-011, 12 iso sheets) from existing 12" header valves to temporary flare
  • Install temporary 6" steam piping (TEMP-002, 11 iso sheets) to supply temporary flare
  • Modify flow orifices on 6" and 2" steam control station bypasses
  • Add two new isolation valves on 24" flare header near MS-468P (Line 600039)
  • Rotate existing 12" valve on MS-468P 180 degrees (Tie-In 27)
  • Remove and replace insulation on 6" steam line on flare stack (asbestos survey required)
  • Demolition of existing FFG piping, AI, FG piping per sketches SK004–SK014, SK022 (12 demo drawings)

Top 5 Confidence Blockers

⚠ CRITICAL: Flare stack height / scaffold scope — not dimensioned anywhere in 136 pages. Tie-in plan suggests ~EL 281'-0" working height. Scaffold is potentially the largest single labor item.
⚠ CRITICAL: NDE Standard A61X-80-413 referenced on every isometric sheet and the spec register — not included in the package. Cannot estimate NDE labor without it.
⚠ CRITICAL: Four P&ID sheets are listed as primary design documents but do not appear anywhere in the package body.
⚠ CRITICAL: Asbestos survey status on insulated lines without blue band is unresolved. Could completely change demo scope and cost.
⚠ HIGH: SS socket welds on 1" 300# lines at top of flare stack — most technically demanding welds on the project. Extreme elevation, SS qualification, TIG process, tight access.
